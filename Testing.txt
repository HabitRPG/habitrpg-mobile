For Ubuntu 12.10 or Linux Mint 14
Prerequisites: 
*Local clone of habitrpg-mobile
*Ripple Emulator installed in your browser

1)Install NVM(https://github.com/creationix/nvm)
	1a)git clone git://github.com/creationix/nvm.git ~/nvm
	1b). ~/nvm/nvm.sh
2)Install Node
	2a)nvm install 0.8.22(type this exactly or it will not work)
3)Install Yeoman
	3a)npm install -g yo grunt-cli bower
4)Install local dependencies into habitrpg-mobile
	4a)cd habitrpg-mobile
	4b)npm install && bower install
5)Run habitrpg-mobile
	5a)cd habitrpg-mobile(if not already there)
	5b). ~/nvm/nvm.sh(if not already done)
	5c)nvm run 0.8.22 server.js
	5d)open browser localhost:3003
	5e)run ripple emulator
		5a)turn tooltips off
		5b)disable cross domain proxy
