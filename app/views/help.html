HabitRPG is a habit-building program which treats your life like a Role Playing Game. Level up as you succeed, lose HP as you fail, earn money to buy weapons and armor, compete with your friends.

The website has been around for a year, and has additional features such as guilds, parties, pets, random drops, tags, difficulty settings, etc. We'll pull in features bit-by-bit to the mobile app - but we wanted to get this initial release out there so people can start using it, and hopefully contributing. It's open source (built on AngularJS & PhoneGap, using standard web technologies) so if you have any HTML, CSS, or JavaScript experience, please join us to help make Habit awesome! File any bugs here. Android 2.3.x users - we'll enable for your devices once we can fix these CSS / Angular issues.
